#LSTUDIO
É um projeto feito para facilitar o agendamento de clientes para um studio.
Basta cadastrar os serviços e realizar os agendamentos,
Também é possível realizar a análise dos dados cadastrados.

#Comandos Importantes
git subtree push --prefix backend origin backend # subir versão do backend pro heroku
ng deploy --base-href=https://leooliveiraz.github.io/lstudio/ # subir versão do frontend pro ghpages do projeto
